school iXiMAKER Community - LEARN

Interface de création et d'édition pour les formations au standard iXi

Bienvenue dans iXiMAKER Community - LEARN

Cet éditeur vous permet de créer et de modifier vos formations XR.
Utilisez les onglets ci-dessus pour naviguer entre les différentes sections de l'éditeur.

  • school Formation : Définissez les paramètres généraux et spécifiques de votre formation.
  • bookmark_star Topics : Ajoutez, modifiez et organisez les sujets et les quizz de votre formation.
  • book Cours : Gérez vos courses en organisant leurs topics.
  • swap_vert Import/Export : Importez des fichiers JSON existants ou exportez votre formation.

⚠️ Les formations LEARN ne peuvent pas fonctionner sans leur procédure WORK !

task_alt Préparation

Avant de commencer la création de votre procédure, vérifiez les points suivants :

counter_1 Création des dossiers

  • Créez votre structure de dossiers comme indiqué, ci-dessous, dans la section "Aide"

counter_2 Préparation des médias

  • Préparez vos médias et copiez-les dans le dossier "XXXX_FR/learn"
    • Images : format jpg, png ou gif (Taille conseillée : 2048px)
    • Audios : format ogg ou mp3
    • Modèles 3D : format glb ou gltf avec textures incluses (10Mo max - 4Mo conseillé)

    ⚠️ Les noms de fichiers et de dossier ne doivent pas contenir de caractères spéciaux, ni accentués tels que : (&é~#à%*$ê/!_\)

info Pour les modifications

Si vous souhaitez éditer un fichier existant :

  • Cliquez sur l'onglet "Import/Export"
  • Importez votre json LEARN via le bouton "Importer" (pas de WORK)
  • Suivez les indications de la section "Pour commencer"

⭐ Pour commencer

  • Cliquez sur l'onglet "Formation" pour définir les paramètres de base.
  • Cliquez sur l'onglet "Topics" pour créer vos sujets et vos quizz
  • Cliquez sur l'onglet "SCours" pour organiser vos sujets en leçons.
  • Cliquez sur l'onglet "Import/Export" pour sauvegarder votre json
    (⚠️le nom doit être exactement le même que votre dossier de référence, avec l'extension .json. (ex: XXXX_FR_learn.json))

👉N'oubliez pas de sauvegarder régulièrement votre travail

  • Utilisez le bouton en bas de page : ▶ Générer JSON
  • Rendez-vous dans l'onglet "Import/Export"
  • Cliquez sur 📤 Exporter puis l''onglet "Import/Export".

🛟 Aide

Fonctionnement

iXiMAKER Community est un éditeur visuel pour vous assister dans l'édition et la création de fichiers JSON iXi.
🆗 Il permet :

  • de simplifier la création de formations XR, en association avec sa procédure WORK
  • d'y assiocer les liens vers les fichiers média
  • de créer des fichiers JSON LEARN conformes au format iXi
  • d'importer et de modifier des fichiers JSON conformes au format iXi

🚫 Il ne permet pas :

  • de copier un fichier media dans le dossier (uniquement version pro)
  • de spatialiser vos contenus (c'est le rôle de "iXiWORK - Place" 😉)
  • de créer un dossier de contenu (uniquement version pro)

Structure des fichiers au standard iXi

Pour fonctionner, les dossiers et fichiers doivent respecter l'organisation suivante :

  • 📁Machine XYZ (Votre "Référentiel" contenant vos procédures et vos formations dans les différentes langues)
    • 📁Machine XYZ_FR (Variante linguistique définie selon le champ "Code pays")
      • Voir l'aide d'iXiMAKER WORK pour plus d'informations sur les fichiers de ce contenu.
      • 📁learn (contenant les fichiers images, sons, modèles 3D référencés dans la formation)
        • 📄Machine XYZ_FR_learn.json (Le json de la formation, contenant les sujets, les quizz et leur organisation)
        • 🧊les_autres_modeles.bytes (les modèles 3D, différents du modèle principal, au format .glb ou .gltf, renommés en .bytes)
        • ...
        • 🖼️les_images.png
        • ...
        • 🔊les_audio.ogg
        • ...

    • Nommage des fichiers

    Pour les noms de fichiers, et afin d'être pleinement compatible avec la version PRO d'iXiMAKER, il est conseiller d'utiliser la syntaxe suivante :

    • 🧊 Hologrammes :
      • topic-4_hologram.bytes : pour l'hologramme du sujet 4
    • 🖼️ Images :
      • topic-1_img.png : pour l'illustration du sujet 1
      • topic-2_env.jpg : pour l'environnement 360° du sujet 2
    • 🔊 Audio :
      • topic-3_aufio.ogg : pour l'audio du sujet 3
