# iXiMAKER-Community
## Editeurs de contenus pour iXiWORK.
Ces 2 éditeurs permettent de créer et modifier des contenus XR au standard iXi et :
- s'utilisent directement avec un simple navigateur internet.
- ne nécessitent pas de connexion
- sont légers (520Ko)
- sont compatibles avec iXiMANAGER et sa version PRO d'iXiMAKER
## Utilisation
1. Décompressez l'archive 
2. Lancez iXiMAKER-Work.html pour créer vos procédures (process et cours théoriques)
3. Lancez iXiMAKER-Learn.html pour créer vos formations (exercices et quizz spatialisé)
4. Spatialisez vos contenus avec iXiPLACE, disponible gratuitement sur https://ixiverse.com
5. Distribuez librement vos contenus XR
## Contenu
- iXiMAKER-Work.html : page web pour la création des JSON WORK
- iXiMAKER-Learn.html : page web pour la création de JSON LEARN
- Demo.zip : exemple de strucutre des fichiers
